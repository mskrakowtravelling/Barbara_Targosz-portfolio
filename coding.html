Weather App

As a final project of my SheCodes Plus workshop I designed and coded a fully functional weather app using Java Script and Axios. It was styled in CSS with elements of Bootstrap. The source of weather data is OpenWeather.

You can use the searchbar to chose the city or you can use your current location to display the weather. Both current weather and 5 days forcast are displayed on the page. It also shows current date and time of the given location as well as your current date and time. It is hosted on a free platform, therefore, it doesn't have a unique web address.

The original project isn't responsive, however, I'm planning to make necessary changes in the nearest future.
